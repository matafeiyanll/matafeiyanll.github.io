Research Interests

Software security, Database, LLM Quality Assurance and Third-party ecosystem.
Currently, I am researching database security and reliability using software testing techniques such as metamorphic testing, while also exploring LLM-aided automation for DBMS testing across diverse SQL dialects. Additionally, I explore the security and quality assurance aspects of LLMs, such as correctness and compatibility issues in code generation.
